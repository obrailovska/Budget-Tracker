# Budget-Tracker

Budget Tracker is an app that allows you to track your expenses and income.

## Installation

- Clone the repo
- run ```npm i``` to install all dependencies
- run ```nmp start``` to start the server and run the application locally 

## Built with: 
- JavaScript
- HTML
- CSS
- MongoDB
